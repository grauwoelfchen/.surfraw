# .Surfraw

This is a repo of personal config files and scripts for [Surfraw](
https://gitlab.com/surfraw/Surfraw).


## Usage

```zsh
% sr rust -dt=ref
% sr rust --toolchain=stable-x86_64-unknown-linux-gnu --document-type=std
```
